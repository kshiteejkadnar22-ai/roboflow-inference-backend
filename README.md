# Roboflow Inference Backend

This repository contains the Python backend for real-time webcam-based
object detection using the Roboflow Inference SDK.

## Requirements
- Python 3.9+
- Webcam
- Roboflow API Key

## Run Instructions
```bash
pip install -r requirements.txt
export ROBOFLOW_API_KEY=your_api_key_here
python app.py
